# SQL Sales Data Analysis
Business data analysis project using SQL queries to analyze sales performance and customer behavior

This project demonstrates business data analysis using SQL.

The goal of the project is to analyze sales performance, product profitability, and customer segments using structured queries.

## Business Questions Answered

• Which products generate the highest profit?
• Which regions generate the most sales?
• What are the monthly sales trends?
• Which customer segments generate the most revenue?

## Tools Used

• SQL
• PostgreSQL
• Data Analysis

## Key Insights

• Identified top profitable products
• Analyzed regional sales performance
• Evaluated customer segment revenue contribution
